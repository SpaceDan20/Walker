# Walker 
Walker is an IsaacLab project involving RL tasks for humanoid locomotion. It is a personal project intended for research and learning purposes.

## Roadmap
- Teach a humanoid to balance
- Teach a humanoid to move
- Teach a humanoid barebones locomotion tasks (walk and try not to faceplant immediately)
- Teach a humanoid basic locomotion
- Teach a humanoid to run
- Teach a humanoid to copy .FBX inputs (experimental)
- Teach a humanoid to dance
- Teach a humanoid to vacuum
- Teach a humanoid to fold my laundry
- Teach a humanoid to file my taxes
- Teach a humanoid to take over the world (experimental)
- Teach a humanoid to conquer the galaxy (in due time)
